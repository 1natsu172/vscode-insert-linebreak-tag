# insert-br-tag

Very easy to insert `<br>` HTML tag keybind, with Dreamweaver's keybinding style.

* simple keybinding

> `shift+enter`

## Features

Insert `<br>` tag keybind.

##### like this.

```
<br><br><br><br><br><br><br><br>
```

## Requirements

```
"vscode": "^1.16.0"
```

## Usage

##### Please `shift+enter`!!
##### Then insert `<br>` tag!!
