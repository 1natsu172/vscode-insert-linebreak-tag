# insert-br-tag

This is the README "insert-br-tag". 

Very easy to insert `<br>` HTML tag keybind, with Dreamweaver's keybinding style.

> `shift+enter`

## Features

Insert `<br>` tag keybind.

## Requirements

```
"vscode": "^1.16.0"
```

## Usage

Please enter `shift+enter`. Then insert `<br>` tag.
